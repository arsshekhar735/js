Phases of Event Propagation

In JavaScript, events propagate through three distinct phases.

  1. Capturing Phase: During this phase, the event is propagated from the root element down to the target element. It allows us to intercept the event before it reaches the target.
  2. At-target Phase: This phase occurs when the event reaches the target element. The event is processed here, at the exact location where the event was triggered.
  3. Bubbling Phase: In the bubbling phase, the event propagates back up from the target element to the root element, allowing any ancestor elements to handle the event.
